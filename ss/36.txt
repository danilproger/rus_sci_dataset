ПРИМЕНЕНИЕ МЕТОДА АНАЛИЗА ИЕРАРХИИ ДЛЯ ПОДДЕРЖКИ ПРИНЯТИЯ РЕШЕНИЯ ПРИ ВЫБОРЕ ЛЕКАРСТВЕННЫХ ПРЕПАРАТОВ 

Разработана модель поддержки принятия решения для выбора наиболее безопасных лекарственных препаратов. Модель, основанная на методе анализа иерархии, позволяет учитывать индивидуальные особенности конкретного пациента.

 Проблема безопасности лекарственных препаратов сегодня является одной из основных в клинической практике. Фармакологический рынок интенсивно развивается, количество лекарственных средств постоянно растет. Врачу необходимо ежедневно читать в среднем около двадцати медицинских статей, чтобы владеть актуальной информацией по своей специальности. Однако время, которое он может уделить чтению на работе, менее одного часа в неделю. В связи с этим адекватно оценить безопасность лекарственного препарата для конкретного пациента, а также выбрать наиболее безопасный из них становится для врача все более сложной задачей. Как следствие затруднение принятия клинических решений, повышение риска клинических ошибок, необоснованность назначения того или иного препарата. Разработанная авторами статьи методика принятия решения для поддержки выбора препарата актуальна при автоматизации задач выбора врачом наиболее безопасного ЛС для конкретного пациента в условиях информатизации общества и врачебной практики в частности. Принятие врачебного решения традиционно ассоциируют с поиском и связанным с ним перебором вариантов лечения. Теоретически процесс принятия решения сводится к определению множества критериев выбора и альтернатив, алгоритма выбора альтернатив согласно заданным критериям. По данным критериям отбирается подмножество альтернатив для дальнейшего выбора наиболее подходящего ЛС. Эти критерии могут иметь различную значимость, и их приоритет определяется врачомэкспертом для конкретного пациента после проведения соответствующей диагностики. Например, для пациента с нарушением функций почек и печени критерии, соответствующие этим системам и органам, будут иметь наивысший приоритет среди остальных, так как даже малое поражение этих систем в результате применения ЛС для него недопустимо. За основу модели поддержки принятия решения был выбран метод анализа иерархии как позволяющий отбирать альтернативы по нескольким критериям, важность которых различна. Общая схема иерархии задачи по поддержке принятия решения о выборе наиболее безопасного ЛС изображена на рисунке. В общем виде алгоритм выбора наиболее безопасного ЛС по описанной модели поддержки принятия решения состоит из следующих этапов. 1. Оценка вероятности нежелательного влияния A для каждого препарата альтернативы на каждую систему и органы пациента. 2. Расстановка критериев и их важности для каждого органа и системы конкретного пациента. 3. Построение матрицы парных сравнений для критериев On столбцы и строки матрицы критерии On значения элементов матрицы отношения значимости критериев B B. 4. Построение матрицы парных сравнений альтернатив для каждого критерия O второго уровня, в которых сравниваются значения A каждой альтернативы. Столбцы и строки матрицы альтернативы D значения элементов матрицы отношения A A. 5. 6. 7. Сортировка альтернативы по возрастанию значения S. Данную методику поддержки принятия решения планируется использовать при разработке автоматизированной системы поддержки принятия врачебного решения по выбору ЛС. Таким образом, оценивая безопасность применения сразу нескольких ЛС с учетом индивидуальных особенностей пациента, врач получает возможность оценить и принять более объективное решение при выборе наиболее безопасного препарата, тем самым снизить риск возникновения нежелательных побочных реакций от приема ЛС и повысить обоснованность их назначения.
